電腦軟體
我們也透過您電腦上所安裝的電腦軟體 (Windows、Mac 應用程式) 提供服務。
我們所收集的您的資訊
在任何情況下，如果您選擇在使用我們的電腦軟體時提供資訊給我們，我們會使用唯一使用者 ID (UUID) 來唯一識別您的裝置。此 UUID 是藉由結合電腦上的 Windows 安全性識別碼 (SID) 與隨機產生的大型數字產生而成，以便讓我們能夠識別已安裝我們的電腦軟體的該台電腦。您可以關閉電腦的所有訊連科技產品來重設裝置的 UUID，以及使用 Windows 登錄編輯程式在「Computer\HKEY_CURRENT_USER\Software\CyberLink\CBE」移除此登錄機碼。

收集和分析我們應用程式的使用現況有助於改善我們的產品設計、功能開發和錯誤修正。CPIP 計劃收集您使用訊連科技應用程式功能的方式、您的搜尋輸入、檔案名稱、您輸入到訊連科技應用程式的檔案的副檔名。
