Uber 業務合作夥伴。 舉例來說，如果使用者透過合作關係或第三方提供的優惠方案預約某項服務，Uber 可能將特定資料分享給該第三方單位。這可能包含與我們 API 整合的其他服務、平台、App 或網站；車輛供應商或服務；擁有與我們整合的 API 或服務的對象；或者是與優惠/獎勵、競賽或特別服務相關的合作餐廳、商家或其他 Uber 業務合作夥伴及其使用者。
緊急援助服務。我們提供的功能可讓使用者在出現緊急狀況時或在發生特定事件後，能與警方、消防隊和救護車服務分享資料。如需更多資訊，請參閱下方的「選擇與透明度」和「緊急資料分享」。
保險公司。 若使用者已就 Uber 服務向保險公司回報或提出索賠，Uber 會與該保險公司分享特定資料，以調整或處理使用者的保險索賠。
商家或餐廳。 如果訂單收件人在使用者帳戶中新增商家常客會員編號，Uber 會在顧客下訂單時與相關商家或餐廳分享常客會員會員資格和訂單詳細資訊。訂單收件人亦得選擇與特定商家或餐廳分享聯絡資訊和訂單詳細資訊，以接收來自該商家或餐廳的行銷資訊。
一般大眾。使用者透過公開平台 (例如：可供大眾查看的 Uber 部落格和 Uber 社群媒體頁面) 提出的問題或意見，包括使用者提出的問題或意見中所含任何資料。
